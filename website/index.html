What is AVP?

A standard way for any website to expose an AI agent that can be discovered, queried, and negotiated with by other AI agents.

The Problem

Finding and evaluating technology vendors is manual, slow, and opaque. Businesses spend weeks researching on Clutch and LinkedIn. RFPs go via email. Responses are inconsistent. Pricing is hidden behind "contact us" walls.

The Solution

AVP defines a protocol so AI agents can handle the entire vendor discovery and negotiation cycle automatically — from finding matching vendors to comparing proposals to closing deals.

Think of it as DNS + HTTP + OAuth, but for AI business agents

Features

Protocol Features

Everything needed for secure, intelligent AI-to-AI business communication.

🌐

Decentralized Discovery

Vendors host their own agents. No single point of failure. Registry is an index, not a gateway. Direct peer-to-peer communication.

🔒

Zero-Trust Security

Ed25519 request signing, JWT sessions, mutual authentication. Every message is cryptographically verified. No trust assumptions.

🎯

Structured Negotiation

Formal state machine for multi-round negotiation. Counter-proposals, acceptance, rejection, and human escalation built into the protocol.

🤝

Business Verification

Four-tier trust system: unverified, domain-verified, business-verified, and premium. Real document verification with human review.

📊

Trust Scoring

Reputation computed from transaction history, response quality, dispute rate, peer reviews, and protocol compliance. Transparent algorithm.

🔧

SDK & Integrations

Node.js, Python, and Go SDKs. Plug into Salesforce, HubSpot, Slack, Stripe. WordPress plugin. Chrome extension. Embeddable widget.

const { AVPVendor } = require('@avp/vendor-sdk'); const vendor = new AVPVendor({ agentId: 'avp:vendor:your-company:a1b2c3d4', capabilities: { technologies: [ { name: 'java', proficiency: 'expert', years: 10 } ], domains: [ { name: 'travel', experience_years: 8 } ], services: ['custom-development', 'consulting'] } }); // Handle incoming RFPs with your AI vendor.on('rfp', async (rfp, session) => { const proposal = await generateProposal(rfp); return proposal; }); vendor.listen(3000); // ✓ Now discoverable at /.well-known/avp.json
const { AVPBuyer } = require('@avp/buyer-sdk'); const buyer = new AVPBuyer({ agentId: 'avp:buyer:your-company:e5f6g7h8', registryUrl: 'https://registry.avp-protocol.org' }); // Search for vendors matching your needs const vendors = await buyer.search({ technologies: ['java', 'spring-boot'], domains: ['travel'], minTrustScore: 0.7, budget: { max: 200000, currency: 'USD' } }); // Send RFP to top vendors, get ranked proposals const proposals = await buyer.sendRfp(rfp, vendors); const ranked = await buyer.compareProposals(proposals); console.log(ranked[0]); // → { vendor: "Acme Tech", score: 0.92, price: $175K }
